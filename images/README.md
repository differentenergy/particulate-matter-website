# Product Images Folder

Place your Broad purifier product images here with these exact filenames:

- `broad-home-purifier.jpg` - Home/residential air purifier
- `broad-commercial-purifier.jpg` - Commercial system for offices/schools
- `broad-industrial-purifier.jpg` - Industrial solution

**Image specifications:**
- Format: JPG or PNG
- Size: 800x800px minimum (1200x1200px ideal)
- Aspect ratio: Square (1:1) preferred
- File size: Under 500KB each

See IMAGE-GUIDE.md in the root directory for detailed instructions.
